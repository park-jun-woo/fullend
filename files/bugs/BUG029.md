# BUG029 — package-prefix @model 빌드 실패

## 분류
SSaC 설계 결함 (코드젠 버그 아님)

## 증상
zenflow-try06 add05에서 `@put session.Session.Set(...)` 사용 시 `go build` 실패.
WithTx, ctx 주입, 인터페이스 생성, 인자 순서, 타입 변환이 전부 깨짐.

## 근본 원인
`@get`/`@put`/`@delete`는 DDL 모델 전용으로 설계됨. session/cache/file은 DDL이 아닌 인프라 서비스이므로 같은 경로로 처리할 수 없음.

## 해결
Phase051에서 package-prefix @model 문법 폐기. session/cache/file은 `@call` Func으로 전환.
