# BUG023: fullend chain — STML 연결 누락

## 단계
chain

## 재현
```bash
fullend chain ListWorkflows specs/dummys/zenflow
```

출력에 STML이 포함되지 않음.

`specs/dummys/zenflow/frontend/workflows.html`에 `data-fetch="ListWorkflows"` 참조가 있으나 체인 결과에 미표시.

동일하게 `workflow-detail.html`의 `data-action="ActivateWorkflow"`, `data-action="ExecuteWorkflow"`, `data-fetch="GetWorkflow"`, `data-fetch="ListActions"`, `data-action="AddAction"` 등도 체인에서 누락.

## 기대 동작
`data-fetch` 또는 `data-action`으로 operationId를 참조하는 STML 파일이 체인에 포함되어야 한다.

```
  STML       frontend/workflows.html:5    data-fetch="ListWorkflows"
```

## 영향
chain으로 feature 영향 범위를 파악할 때 프론트엔드 의존성이 누락되어 STML 수정 필요 여부를 놓칠 수 있다.
