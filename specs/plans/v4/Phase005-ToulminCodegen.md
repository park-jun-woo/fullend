# Phase005 — codegen 교체: Rego 파일 복사 → Go toulmin 코드 생성

## 목표

`fullend gen`이 Rego 파일을 복사하는 대신, toulmin Graph Builder 기반 Go 인가 코드를 생성한다.

## 배경

현재 codegen:
1. `internal/gen/gogin/authz.go` — `.rego` 파일을 `artifacts/backend/internal/authz/`에 복사
2. `internal/gen/gogin/generate_main.go` — `authz.Init()` 호출 코드 생성 (OPA 로딩)

전환 후:
1. specs의 `authz/*.go` 규칙 함수를 복사 + YAML에서 `graph_gen.go` 생성
2. `main.go`에서 `authz.Init(conn, mappings, graph)` 호출 코드 생성

## 생성 대상 파일

### 1. `artifacts/backend/internal/authz/rules.go`

specs의 authz/rules.go에서 규칙 함수를 복사. import와 package 조정.

### 2. `artifacts/backend/internal/authz/graph_gen.go` (신규 생성)

YAML 그래프 정의에서 Graph Builder 코드 생성. `toulmin graph authz.yaml`의 출력과 동일한 형태.

```go
// Code generated by fullend. DO NOT EDIT.
package authz

import "github.com/park-jun-woo/toulmin/pkg/toulmin"

var AuthzGraph = toulmin.NewGraph("authz").
    Warrant(DenyByDefault).
    Rebuttal(ExecuteWorkflow).
    Rebuttal(AdminAllowAll).
    Rebuttal(OwnerAccess).
    Rebuttal(SuspendedBlock).
    Defeat(ExecuteWorkflow, DenyByDefault).
    Defeat(AdminAllowAll, DenyByDefault).
    Defeat(OwnerAccess, DenyByDefault).
    Defeat(SuspendedBlock, AdminAllowAll).
    Defeat(SuspendedBlock, OwnerAccess).
    Defeat(SuspendedBlock, ExecuteWorkflow)
```

### 3. `artifacts/backend/internal/authz/ownership.go`

specs의 ownership.go를 복사.

### 4. `main.go` 변경

```go
// 기존 생성 코드
if err := authz.Init(conn, []authz.OwnershipMapping{...}); err != nil {
    log.Fatalf("authz init failed: %v", err)
}

// 신규 생성 코드
if err := authz.Init(conn, authz.Ownerships, authz.AuthzGraph); err != nil {
    log.Fatalf("authz init failed: %v", err)
}
```

## 기존 codegen 삭제

| 파일 | 변경 |
|------|------|
| `internal/gen/gogin/authz.go` | `.rego` 복사 로직 → Go 파일 복사 + graph_gen.go 생성으로 전면 교체 |
| `internal/gen/gogin/generate_main.go` | `authz.Init()` 시그니처 변경 (Graph 주입) |

## 생성 코드에서 OPA 흔적 제거

생성되는 `go.mod`에서:
- `github.com/open-policy-agent/opa` 제거
- `github.com/park-jun-woo/toulmin` 추가

생성되는 `backend/internal/authz/`:
- `.rego` 파일 → 없음
- `rules.go` + `graph_gen.go` + `ownership.go`
