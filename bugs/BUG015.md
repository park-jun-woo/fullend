# BUG015: ssac-gen __RESPONSE_STATUS__ 플레이스홀더 미치환 — 빌드 실패

## 단계
ssac-gen

## 심각도
CRITICAL — 전체 빌드 불가

## 증상
`go build` 시 모든 서비스 파일에서 `undefined: __RESPONSE_STATUS__` 컴파일 에러. 18개 서비스 파일 전부 영향.

## 재현
fullend v0.1.4에서 `fullend gen` 실행 후 `go build`.

## 생성된 코드 예시

```go
// internal/service/auth/register.go
c.JSON(__RESPONSE_STATUS__, gin.H{...})  // ← 치환 안 됨
```

## 기대 동작

```go
c.JSON(http.StatusOK, gin.H{...})       // 200 응답
c.Status(http.StatusNoContent)           // 204 응답
```

## 원인
BUG014(204 상태코드 지원) 수정 시 응답 상태코드를 템플릿 변수 `__RESPONSE_STATUS__`로 변경했으나, 치환 로직이 누락됨.

## 수정 제안
템플릿 렌더링 시 `__RESPONSE_STATUS__`를 OpenAPI 응답 코드에 따라 `http.StatusOK`, `http.StatusNoContent` 등으로 치환.
