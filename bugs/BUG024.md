# BUG024: Claims *_id 필드가 string으로 생성됨

## 원인
Phase030에서 claims 타입 힌트 형식이 `claim_key:go_type`으로 변경되었으나, 매뉴얼이 구 형식(`*_id → int64` 자동 추론)으로 남아있었음.

## 올바른 사용법
```yaml
claims:
  ID: user_id:int64       # 명시적 타입 힌트 필요
  Email: email            # 타입 미지정 → string (기본값)
  Role: role
```

## 조치
매뉴얼(`artifacts/manual-for-ai.md`) 업데이트 완료.
